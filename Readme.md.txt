# Shadow of the Erdtree Guides

![Banner](assets/images/banner.jpg)

Unofficial fan-made guides for Elden Ring's Shadow of the Erdtree DLC.

## Features
- Boss strategies and weaknesses
- Weapon and build recommendations
- Location walkthroughs
- Interactive maps
